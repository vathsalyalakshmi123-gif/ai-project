# Employee Onboarding Milestone Tracker

A professional React onboarding tracker for multi-stage employee checklists, with backend persistence and employee progress management.

## Features

- Multi-stage onboarding milestones
- Employee selection and stage navigation
- Task completion checklists with instant save
- Backend API and JSON persistence
- Professional responsive UI built with React and Vite

## Setup

1. Open a terminal in the project folder.
2. Run `npm install`.
3. Run `npm run dev` to start the frontend development server.
4. Run `npm start` after `npm run build` to serve the app with the backend.

## Scripts

- `npm install` — install backend and frontend dependencies
- `npm run dev` — start Vite development server
- `npm run build` — build frontend assets
- `npm start` — start Express backend server

## Backend API

- `GET /api/progress` — load onboarding data
- `POST /api/progress` — save task status and current stage
